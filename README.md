# 🏦 Loan Defaulter Predictor — FastAPI Web App

A full-stack web application that predicts whether a bank customer is likely to
default on a loan, built on top of the Logistic Regression model from the
`Project_Defaulter_in_loans.ipynb` notebook.

---

## 📁 Project Structure

```
loan_defaulter_app/
├── main.py                  ← FastAPI app (routes + prediction logic)
├── train_model.py           ← One-time model training script
├── requirements.txt         ← Python dependencies
├── credit_risk_dataset.csv  ← ⚠️ YOU must place this here
├── model.pkl                ← Generated by train_model.py
├── scaler.pkl               ← Generated by train_model.py
├── templates/
│   └── index.html           ← Beautiful dark-theme UI
└── static/                  ← (empty, reserved for custom assets)
```

---

## 🚀 Setup & Run

### 1. Place your dataset
Copy `credit_risk_dataset.csv` into this folder (same directory as `main.py`).

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model (run ONCE)
```bash
python train_model.py
```
This creates `model.pkl` and `scaler.pkl`.

### 4. Start the web server
```bash
uvicorn main:app --reload
```

### 5. Open in browser
```
http://127.0.0.1:8000
```

---

## 🌐 API Endpoints

| Method | URL | Description |
|--------|-----|-------------|
| GET  | `/`          | Web UI (form) |
| POST | `/predict`   | Form submission → HTML result page |
| POST | `/api/predict` | JSON REST API |

### REST API example (curl)
```bash
curl -X POST http://127.0.0.1:8000/api/predict \
  -H "Content-Type: application/json" \
  -d '{
    "age": 35,
    "income": 50000,
    "loan_amount": 120000,
    "credit_score": 580,
    "employment_years": 3,
    "education_level": "Bachelors",
    "housing_status": "Rent"
  }'
```

### Response
```json
{
  "prediction": 1,
  "is_defaulter": true,
  "default_probability": 72.4,
  "safe_probability": 27.6,
  "risk_level": "High"
}
```

## 📊 Model Details

| Item | Value |
|------|-------|
| Algorithm | Logistic Regression |
| Class weight | balanced (handles imbalanced data) |
| Scaler | StandardScaler |
| Features | Age, Income, Loan Amount, Credit Score, Employment Years, Education Level, Housing Status |

---

## 🔧 Features Used

| Feature | Type | Notes |
|---------|------|-------|
| Age | Numeric | Years |
| Income | Numeric | Mean-imputed for missing values |
| Loan Amount | Numeric | — |
| Credit Score | Numeric | 300–850 |
| Employment Years | Numeric | — |
| Education Level | Ordinal | High School=0, Bachelors=1, Masters=2, PhD=3 |
| Housing Status | One-hot | Mortgage (base), Own, Rent |
